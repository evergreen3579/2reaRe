1. 시스템 > 정보 > 고급 시스템 설정 > 환경 변수

2. 환경 변수 > 시스템 변수 > 새로 만들기
   변수 이름 : JAVA_HOME
   변수 값 : 다운로드한 JDK17 폴더 경로
   ( 예 : C:\Program Files\Java\jdk-17 )

3. 환경 변수 > (사용자명)에 대한 사용자 변수 > Path선택 > 편집
   새로 만들기 : " %JAVA_HOME%bin " 입력 ( 맨 위로 이동 )

4. 확인을 눌러 환경변수 설정 저장

5. cmd 창 열기
   " java -version " 입력 후 버전 확인
